## PONTOS TÉCNICOS IMPORTANTES PARA DESTACAR NO VS CODE:

### 1. Sistema de Variáveis CSS (style.css, linhas 6-45)
- Tokens de cor organizados
- Cores primárias (verde) e secundárias (rosa)
- Sistema de nomenclatura consistente

### 2. Estrutura HTML Semântica (index.html)
- Tags semânticas: `<section>`, `<nav>`, `<footer>`
- Estrutura bem organizada
- Acessibilidade com alt texts

### 3. Componentes Modulares
- Navbar separada (navbar.css)
- Botões reutilizáveis (button.css)
- Footer consistente em todas as páginas

### 4. Organização de Arquivos
- Pastas separadas: css/, components/, pages/, assets/
- Nomenclatura clara e consistente
- Estrutura escalável

### 5. Responsividade
- Media queries implementadas
- Layout flexível
- Design mobile-friendly
